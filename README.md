## 🎯 What It Does

ChatterBuddy provides AI-powered emotional and cognitive wellness support for elderly users. The platform consists of:
- **Public marketing website** introducing the product to seniors and families
- **Admin portal** for aged care providers to manage residents, accounts, and subscriptions
- Integrated companion app (mobile, separate stack)

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js, TypeScript, Tailwind CSS |
| Backend | Node.js, REST APIs |
| Cloud | AWS (S3, EC2) |
| Auth | JWT + Role-based access control |

## ✨ Key Features

### Web Platform
- 📱 Marketing site with conversion-optimized CTAs
- 📝 Subscription signup flow for individuals and aged care facilities
- 🔐 User account creation and verification

### Admin Portal
- 🏢 **Multi-account management** — aged care providers can manage hundreds of resident accounts from one dashboard
- 👥 **Role-based admin access** — facility admins, care coordinators, super-admins
- 💼 **Subscription management** — plan tiers, billing oversight, account status
- 📊 **Centralized administration** for aged care providers
- 📋 Resident profile management with care notes

## 🏗 What I Built

- Designed and built the full web platform from scratch (marketing site + signup flows)
- Architected the admin portal with multi-tenant data isolation
- Implemented role-based access control (RBAC) for aged care facility hierarchies
- Set up subscription management UI with billing integration
- Built reusable component library shared between web and portal
